第二章 方法

2.1 方法的定义格式

  • 方法就是若干语句的功能集合（工厂）
  • 参数（原料）
  • 返回值（产物）

:

修饰符 返回值类型 方法名称（参数类型 参数名称,...）{
    方法体;
    return 返回值;
}

  • 修饰符： public static

  • 返回值类型：void

  • 方法名称：main

  • 参数类型：String[]

  • 参数名称：args

  • return:停止当前方法,将返回值返回至调用处;

return返回值必须和方法定义的返回值类型对应

2.2 方法的三种调用格式

  • 单独调用

方法名(参数);

  • 打印调用

System.out.println(方法名(参数));

  • 赋值调用

数据类型 变量名称=方法名(参数);

2.3 有无参数

是否要求数据条件

2.4 有无返回值

2.5 方法使用的注意事项

  • 方法的执行一定要调用
  • 返回值为void时，可以写return;
  • 一个方法中可以有多个return，但要确保只有一个会执行
  • 方法应该定义在类中，方法不能定义在方法中

    2.6 方法的重载（overload）

  • 多个方法的名称相同（Java中严格区分大小写），但是参数列表不同

    println即调用了多种数据类型的重载。

  • 方法重载与下列因素相关：
    • 参数个数不同
    • 参数类型不同
    • 参数的多类型顺序不同
  • 方法重载与下列因素无关：
    • 与参数的名称无关
    • 与方法的返回值类型无关
    • 和修饰词无关

第三章 数组

3.1 数组的概念

  • 可存放多个数据值的容器
  • 特点：
    • 数组是一种引用数据类型
    • 多个数据值的数据类型必须统一
    • 数组的长度在程序运行期间不可改变

3.2 数组的定义格式

  • 初始化：在内存中创建一个数组，并且向其中赋予一些默认值

动态初始化（直接指定数组的长度）

数据类型[] 数组名称 =new 数据类型[数组长度(int)];

静态初始化（直接指定数组的内容）

标准格式：数据类型[] 数组名称 =new 数据类型[]{xx,xx,xx,xx…};
省略格式：数据类型[] 数据名称 ={xx,xx,xx,xx…};

静态初始化没有直接指定长度，但是仍然会自动推算出长度。

静态初始化标准格式可以拆分为两个步骤

int[] array;

array=new int[]{1,2,3};

动态初始化也可以拆分为两个步骤

int[] array;

array=new int[10];

只有带new的定义格式可以拆分为两个步骤

不确定数组中具体内容是用动态初始化，已经确定内容的有静态初始化

3.3 访问数组元素进行获取或赋值

int[] array ={10,20,30};
System.out.println(array);
//[I@723279cf 得到数组对应的内存地址哈希值
[(代表这是一个数组类型)I(里面存放的是int类型)@723279cf(十六进制数)

访问数组元素的格式：

数组名称[索引值(排序编号,从 零 开始到 数组长度-1 )]
array[0]=10;
array[2]=30;
  • array[1]=123//赋值
  • 使用动态初始化数组时，会具有默认值
    • 整型为0
    • 浮点型为0.0
    • 字符型为’\u0000’(\u代表unicode)
    • 布尔型为false
    • 引用类型为null

静态初始化时也有默认值，但系统会自动替换为大括号中的具体值

3.4 常见问题

数组索引越界异常

int[] array ={1,2,3};
System.out.println(array[3]);
//Exception:ArrayIndexOutOfBoundsException

空指针异常

int[] array=null;
//所有引用类型变量，都可以赋值为一个null值，代表其中什么都没有
System.out.println(array[0]);
//Exception:NullPointerException

3.5 数组补充

获取数组长度 

int[] array={xx,xx,xx,xx...}
int len=array.length;

数组一旦创建，程序运行期间，长度不可改变。

int[] array=new int[3];
System.out.println(array.length);//3
array=new int[5];
System.out.println(array.length);//5
//array变成了一个新的数组！！之前的数组被回收了！！

数组的遍历输出

int[] array={xx,xx,xx,xx...};

for(i=0;i<array.length;i++)
{
    System.out.println(array[i]);
}

//array.fori

求数组最大值

int[] array={xx,xx,xx...};
int max=array[0];
for(i=0;i<array.length;i++)
{
    if(array[i]>max) max=array[i];
}

数组的逆序

int[] array={xx,xx,xx...};
for(int min=0,max=array.length-1;min<max;min++,max--)
{    
    int tmp=array[min];
    array[min]=array[max];
    array[max]=tmp;
}

数组作为方法参数

public static void main(String[] args){
    int[] array={xx,xx,xx...}
    printArray(array);
}
public static void printArray(int[] args){
    for(int i=0;i<array.length;i++){
        System.out.println(array[i]);
    }
}
//传参时传的是数组的地址值

数组作为返回类型

public static void main(String[] args){
    int[] result= calculate(1,2,3);
}
public static int[] calculate(int a ,int b,int c){
    int sum=a+b+c;
    int avg=sum/3;
    int [] array=new int[2];
    array[0]=sum;
    array[1]=avg;
    return array;
}
//返回的是数组的地址值

任何数据类型都可以作为方法的参数类型或者是返回值类型
